# 🎬 Movie site 

Web stranica za prikaz 20 američkih filmova kroz tablicu, galeriju i grafikon.

## Funkcionalnosti
- tablica filmova  
- klik na film → detalji (poster + opis)  
- galerija postera  
- CSS grafikon žanrova  
- responzivan dizajn
