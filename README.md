# 🎬 LV1 – Web programiranje: Filmovi

Ovaj projekt predstavlja responzivnu web stranicu o američkim filmovima izrađenu pomoću HTML-a i CSS-a, bez korištenja JavaScripta.

## Web stranica

https://lucpuc244.github.io/lv1-web-filmovi/

## GitHub repozitorij

https://github.com/lucpuc244/lv1-web-filmovi

## Opis projekta

Stranica sadrži:

* Tablicu s 10 američkih filmova
* Hover efekt – prikaz postera filma
* Klik na naslov – prikaz detalja o filmu (slika + opis)
* Grafikon žanrova izrađen pomoću CSS-a
* Galeriju slika filmova
* Responzivan dizajn za različite uređaje

##Tehnologije

* HTML5
* CSS3
* Git & GitHub
* GitHub Pages (deploy)

## Napomena

Interaktivnost (hover i klik efekti) ostvarena je isključivo pomoću CSS pseudo-klasa (`:hover` i `:target`), bez JavaScripta.
